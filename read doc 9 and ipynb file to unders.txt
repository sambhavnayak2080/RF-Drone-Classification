Context Summary (for adaptation)
The spectrogram images were generated from complex I/Q bursts using:
* DC removal & Gram-Schmidt correction
* AWGN & impulse noise augmentation
* FFT size: 2048
* Log power spectrum
* Percentile normalization
* Viridis colormap
* Resized to 224×224 RGB
The training pipeline must adapt the VTI methodology to this spectrogram dataset and the new multitask objective.
Model 4: Xception (Depth-Separable CNN)

RF Spectrogram Adaptation:
- Depthwise separable convolutions decouple spatial (frequency-time) and channel-wise learning
- Efficient for RF where frequency patterns (spatial) and signal characteristics (channels) are independent
- 36 convolutional layers with residual connections prevent gradient degradation
- Modification: Global average pooling preserves spatial translation invariance


Model 5: Vision Transformer (ViT-Small/16)

RF Spectrogram Adaptation:
- Global self-attention captures long-range frequency dependencies across entire spectrogram
- 16x16 patches create 196 tokens, each representing a local frequency-time region
- Positional embeddings encode spatial relationships between RF patterns
- Modification: Small variant (22M params) with attention extraction for explainability


Model 6: Swin Transformer Tiny

RF Spectrogram Adaptation:
- Shifted window attention balances local RF pattern extraction with cross-region interaction
- Hierarchical feature maps (56->28->14->7) capture multi-scale frequency structures
- Linear complexity O(n) vs ViT O(n^2) enables efficient processing
- Modification: Tiny variant (28M params) provides transformer SOTA with manageable compute
